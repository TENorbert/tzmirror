Mirror of the Time Zone Database

On 2011-09-30, some astrology(!) company, "Astrolabe, Inc. v. Olson et al", filed a lawsuit against the people maintaining the time zone database located at ftp://elsie.nci.nih.gov/pub, claiming they had bought the rights to The American Atlas, where some of the data origins. The result of this lawsuit was that the FTP server and TZ mailing list were shut down at 2011-10-06.

Actions like this can't be tolerated, no one can claim ownership of something as essential as timezone data. What next? Patenting the alphabet?

This is a mirror of the timezone files made available for download. The original files (as of 2011-09-26) are stored in the tzmirror/ directory.

More info about this madness is available here.

Files

  • tzmirror/ - The actual files
  • tzmirror.git - Git repository of the files
  • tzmirror.tar.gz - Archive containing all files
  • tzdata.git - Git mirror of the svn repository at http://code.google.com/p/tzdata/
  • tzmirror.sha1 - File with SHA1 checksums of all files

How to clone the data with Git

The files can also be downloaded with Git, to make it easier to download and push to other mirrors.

  • git clone http://tzmirror.sunbase.org/tzmirror.git
  • git clone http://tzmirror.sunbase.org/tzdata.git

To clone this web page:

  • git clone http://tzmirror.sunbase.org/.git
