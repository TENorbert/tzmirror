Mirror of the tz Database

Last updated: Tue, 11 Oct 2011 15:51:44 UTC

On 2011-09-30, an astrology(!) company called "Astrolabe, Inc." filed a lawsuit against the maintainers of the time zone database, claiming the database infringed their copyright. The result of this lawsuit was that the FTP server and tz mailing list were shut down at 2011-10-06.

Actions like this can't be tolerated, no one can claim ownership of something as essential as timezone data. What's next? Patenting the alphabet?
More info about this madness is available here.

This is a mirror of the tz database files and version control repositories.

Files

  • tzfiles/ - This directory contains the original timezone files from the tz FTP server (updated 2011-10-10)
  • tzfiles.sha1 - Contains SHA1 checksums of all timezone files
  • tzfiles.sha1.asc - GnuPG signature of the SHA1 file

Repositories

  • https://gitorious.org/tz/tzfiles - Git repository of the tz database files - current version and archive
  • https://gitorious.org/tz/tzdata - Git mirror of the defunct tzdata svn repository at http://code.google.com/p/tzdata/

How to clone the data with Git

  • git clone https://git.gitorious.org/tz/tzfiles.git
  • git clone https://git.gitorious.org/tz/tzdata.git

News

  • 2011-10-11 — The Subversion repository at Google Code, which hosted the tz data in various formats (CSV, JSON, XML, etc), has disappeared. Fortunately, three days before this happened, the repository was converted to Git, and can now be cloned from Gitorious.
  • 2011-10-10 — A new version (tzdata2011l.tar.gz) of the tz database is released
  • 2011-10-08 — This mirror is created. Mirror skeleton stored at Github, the actual files are stored outside the US, on Gitorious.
  • 2011-10-06 — The main FTP server at elsie.nci.nih.gov is shut down.
  • 2011-09-30 — Astrolabe files the lawsuit.
